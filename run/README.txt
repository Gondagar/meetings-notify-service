Dependencies

1) Installed Java 8. Link https://www.oracle.com/java/technologies/javase-jre8-downloads.html
2) Windows - You need to add a directory with java.exe to classpath. Detail https://docs.oracle.com/javase/tutorial/essential/environment/paths.html
   or it will be enough to go to the catalog where it was installed Java. Default path in Windows 10  C:\Program Files\Java\jdk1.8.0_191\bin  and run start scrypt from this directory.
   In Mac and Unix OS Only installed Java is enough. For install write in terminal 'sudo apt install default-jre'
